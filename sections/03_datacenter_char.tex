\begin{frame}{Длительность выполнения задач}
    Средняя длительность выполнения задач в \textbf{2.7--12.8×} \textit{короче}, чем в других DL-трейсах.\\
    Причины:
    \begin{itemize}
        \item Много небольших задач для GPU.
        \item Одна задача использует всего от 5-20 GPU.
        \item Много задач, которые нельзя выполнить.
    \end{itemize}
    \begin{center}
        \includegraphics[width=0.7\textwidth]{"figures/job_duration_cdf.png"}\\
        {\tiny среднее выполнение задачи на GPU (минуты)}
    \end{center}
\end{frame}

\begin{frame}{Нагрузка на GPU}
    \begin{itemize}
        \item LLM - вычислительно зависимые системы.
        \item Много задач не доходит до GPU, отсеиваясь еще на этапе инициализации.
    \end{itemize}
    \begin{center}
        \includegraphics[width=0.7\textwidth]{"figures/median_utilization.png"}\\
        {\tiny среднее использование GPU}
    \end{center}
\end{frame}

\begin{frame}{Распределение нагрузки}
    \begin{itemize}
        \item Половина задач может выполнятся только на 1 GPU.
        \item Большую часть времени занимают задачи, где требуется много GPU.
    \end{itemize}
    \begin{center}
        \includegraphics[width=0.7\textwidth]{"figures/workload_dist.png"}\\
        {\tiny распределение нагрузки в зависимости от количества запрошенных GPU}
    \end{center}
\end{frame}

    \begin{columns}
